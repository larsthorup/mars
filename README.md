Platform: NodeJS

Libs: Restify, Knex, SQLite

Tools: Mocha, Sinon, Istanbul, JSHint

Quick start:

    npm run dev

Install:

    npm install

Run static analysis:

    npm run lint

Run tests:

    npm run test

Report coverage:

    npm run cover
