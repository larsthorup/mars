mars
====

[![Build Status](https://travis-ci.org/larsthorup/mars.png)](https://travis-ci.org/larsthorup/mars)

Libs: NodeJS, Knex, Restify
Tools: Mocha, Sinon, Istanbul, JSHint

Quick start:

    npm run dev

Install:

    npm install

Run static analysis:

    npm run lint

Run tests:

    npm run test

Report coverage:

    npm run cover
